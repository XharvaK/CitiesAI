## Reddit (r/CitiesSkylines2 or r/CitiesSkylines)

**Title:** `[Beta] CitiesAI: local dashboard + AI advisor that reads your live city stats (Windows)`

**Body:**

I'm looking for beta testers for **CitiesAI**, a free desktop companion I built for CS2.

**What it does**

- Installs a small **read-only** export mod that writes city metrics to a JSON snapshot every ~10s while you play
- Local dashboard: population, treasury, wellbeing, traffic, and **Issues** for live city pressures (water, health, jobs, transit)
- **Ask** tab: questions grounded in your numbers + CS2 wiki / in-game encyclopedia
- Optional AI answers via your own free Mistral API key (stats work without it)

**What you need**

- Windows 10/11, CS2 (Steam or Game Pass)
- No Python, Unity, or modding toolchain. Single installer.

**Beta caveats**

- Unsigned installer (SmartScreen warning)
- Windows only for now
- I'll share the download link via DM / comment to keep volume manageable

If you're interested, comment or DM and I'll send the installer + short setup guide. Feedback button is built into the app. Bugs, wrong answers, and UX notes all welcome.
